3️⃣ AI Processing Layer

This layer uses LLM intelligence.

Modules

Intent Detection

Understands employee request.

Example

Input:
"I want to work from home next week"

Intent:
WFH request
Request Structuring

Extracts structured data.

Example

{
  request_type: "Work From Home",
  start_date: "2026-03-20",
  end_date: "2026-03-25",
  reason: "caregiving"
}
AI-Generated Justification

Converts informal reasoning into professional text.

Example

Employee text:
"My kid is sick"

AI output:

"The employee is requesting a temporary
work-from-home arrangement due to
caregiving responsibilities."
Policy Matching / Tagging

AI matches request with relevant policies.

Example

Matched Policies

• Flexible Work Policy
• Caregiver Support Policy

Technique

RAG (Retrieval Augmented Generation)

11. Architecture Flow Example

Step 1

Employee sends message

"I need shift change next week"

Step 2

Chatbot sends request to backend.

Step 3

AI extracts request details.

Step 4

AI generates professional justification.

Step 5

System tags policies.

Step 6

Workflow engine sends to manager.

Step 7

Manager approves via chatbot.

Step 8

System tracks outcome and feedback.
